Proyecto: Generador de Libros Educativos con IA (TEI)
------------------------------------------------------

Este sistema genera libros educativos en formato PDF con:
✔ Contenido estructurado por capítulos
✔ Ilustraciones generadas por inteligencia artificial
✔ Diseño educativo y adaptado a edades

Cómo ejecutarlo:
----------------
1. Crear y activar un entorno virtual:
   python -m venv .venv
   .venv\\Scripts\\activate (en Windows)

2. Instalar dependencias:
   pip install -r requirements.txt

3. Crear un archivo .env en la raíz con esta línea:
   OPENAI_API_KEY=TU_API_KEY_AQUI

4. Ejecutar:
   python main.py
